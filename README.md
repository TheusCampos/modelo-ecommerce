# Projeto Web Post 06

Este projeto foi estruturado seguindo as melhores práticas de desenvolvimento Front-End.

## Estrutura de Pastas

A estrutura segue o padrão obrigatório:

- `assets/css`: Estilos organizados por escopo (global, pages, partials).
- `assets/js`: Scripts modulares (global, pages, partials).
- `assets/imagens`: Recursos visuais.
- `assets/favicons`: Ícones do site.

## Como Rodar

1. Clone o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Para desenvolvimento, recomenda-se usar um servidor local (ex: Live Server do VS Code ou `http-server`).

## Tecnologias

- HTML5 Semântico
- CSS3 (Variables, Grid, Flexbox)
- JavaScript ES6+ Modules
